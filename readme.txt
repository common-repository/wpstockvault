Easily add images from stockvault.net via shortcode

== Description ==

With wpStockvault you can very easily display images from stockvault.net via shortcode.

There are various ways to customize the output which is described in the Guide that can be found in the Admin Menu

== Installation ==

1. Upload `index.php` to the `/wp-content/plugins/wpstockvault` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
* or
1. Search for wpstockvault via Plugins - Add New in the admin area
2. Click on Install
3. Activate plugin

== Frequently Asked Questions ==

= What is stockvault.net =

Stockvault.net is a website that has free to use images. These images are uploaded by various users.
